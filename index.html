📊 Koneksi Google Spreadsheet (Database)

Google Sheets = Database Anda! Semua data produk & transaksi disimpan langsung di Google Spreadsheet. Ikuti langkah berikut:

  1. Buat Google Spreadsheet baru
  2. Klik menu Extensions → Apps Script
  3. Hapus semua kode, lalu paste isi file Code.gs
  4. Klik Deploy → New deployment
  5. Pilih type: Web app
  6. Set "Execute as" = Me
  7. Set "Who has access" = Anyone
  8. Klik Deploy, copy URL yang muncul
  9. Paste URL tersebut di form di bawah ini
URL ini didapat setelah deploy Apps Script sebagai Web App
